## 开发方式

- 前端开发（vue）

    1. 进入 src 目录：

       `cd ./sd-webui-prompt-all-in-one/src`

    2. 初始化node环境：

       `npm install`

    3. 修改代码

    4. 编译：

       `npm run build`

    5. 重启 stable-diffusion-webui。

- 后端开发（python）

    1. 进入扩展目录：

       `cd ./sd-webui-prompt-all-in-one`

    2. 修改代码

    3. 重启 stable-diffusion-webui。
